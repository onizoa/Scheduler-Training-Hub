Xtime Scheduler Trainee Guide: Your Essential Workflow

Welcome! This guide outlines the core functions and procedures you will use daily in the Xtime Scheduler to manage service appointments, customer records, and shop capacity.

Start Learning Workflow

Core Training Modules

1. System Access and Setup

Loggin In: You will begin by using your unique credentials to log into the system. Access is secure and role-based. This is your first step before performing any action.

Dealership Selection: If your login is associated with multiple locations, your first step will be to select the active dealership from the dropdown menu. This ensures you are viewing and scheduling against the correct shopâ€™s capacity.

2. Appointment Management

New Appointment: This is the primary function for creating a new booking. You will use this tool to search for customer records, select the correct vehicle, define the services needed, and assign the appointment to an available time slot based on real-time shop load.

Rescheduling: When a customer needs to change an existing appointment, use the Rescheduling function. This is critical for moving the booking to a new time slot and immediately freeing up the previously reserved capacity.

Canceling: If an appointment is terminated, use the Canceling function to completely remove the booking from the schedule and instantly return that capacity back to the available inventory.

3. Customer and Vehicle Data

New Customer: If a customer does not exist in the database, use this module to create a new profile. Accurate data entry here is essential for all future service interactions.

Add/Delete Vehicle: Use this function to manage the vehicles associated with a customer's profile (e.g., adding a new purchase, deleting an old trade-in). This ensures the correct vehicle information and service history are always referenced during scheduling.

4. Integration and History

DMS Appointment: Xtime maintains a constant connection with our Dealer Management System (DMS). This means appointments booked in Xtime flow directly into the DMS, and vice-versa. Always check the DMS status to confirm synchronization.

RO History (Repair Order History): This module allows you to instantly review a customer's past services, including declined services and recommended future work. This history is vital for providing relevant recommendations when creating a new appointment.

5. Daily Oversight

Workbook: The **Workbook** is your central daily task manager and visual schedule. It provides a comprehensive view of all appointments, walk-ins, and shop load in real-time. You will use the Workbook constantly to monitor the flow of the service drive and manage capacity throughout the day.
